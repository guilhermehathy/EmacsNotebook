Jupyter

Continuando o tema abordado pelo Professor Fernando Mayer no ultimo post. Hoje aboraremos o uso do Jupyter, mais especificamente a juno do Jupyter Notebook com o Emacs.

Mas afinal, o que um Jupyter Notebook?

Muitos alunos do curso de Estatstica esto familiarizado com a utilizao do RMarkdown para a elaborao de trabalhos acadmicos e de forma bem simplificada o Jupyter Notebook faz o mesmo trabalho, integrando cdigo e texto, assim facilitando na escrita de textos cientficos, pesquisas reproduzveis e etc. O Jupyter uma ferramenta muito poderosa, pois nos permite criar documentos utilizando varias linguagens, como R, Julia e Python.

Instalando o Jupyter

Para instalar o Jupyter, basta rodar o cdigo abaixo (aqui utilizando o pip como instalador).

sudo pip3 install -U jupyter

Para abrir um Jupyter Notebook basta rodar o seguinte cdigo no terminal.

jupyter notebook

Instalando o EIN Emacs IPython Notebook

Para utilizar o Jupyter Notebook no Emacs utilizaremos o pacote ein.

A principal motivao para rodar o Jupyter no emacs a utilizao de atalhos e ferramentas do proprio Emacs. Eu por exemplo costumo utilizar muito o pacote multiple-cursors, que permite a edio de varias linhas do nosso cdigo ao mesmo tempo.

A instalao do Ein deve ser feita atravs do MELPA (que o procedimento padro para se instalar qualquer pacote no Emacs). Para habilitar o MELPA (se ainda no estiver habilitado) veja esse link. Feito isso, digite:

M-x package-list-packages

para abrir a lista de pacotes disponveis do MELPA. Procure pelo Ein com

C-s Ein

e aperte Enter para instalar automaticamente. Feito isso, abra seu ~/.emacs e insira essas linhas para habilitar o pacote:

(package-initialize)
(require 'ein)
(require 'ein-loaddefs)
(require 'ein-notebook)
(require 'ein-subpackages)

Aps inserir salve com C-x C-s e reinicie o Emacs.

Iniciando um Notebook no Emacs

Aps a instalao do pacote Ein, podemos comear utilizar o Jupyter Notebook integrado com o Emacs. Para comear devemos iniciar um servidor do Jupyter no nosso computador. Para isso abrimos o nosso Emacs e utilizaremos o comando M-x ein:jupyter-server-start, em seguida devemos colocar o local no qual esta instalado o Jupyter Server Command: /usr/local/bin/jupyter (Pasta padro no Ubuntu) e por ultimo colocaremos o caminho para a pasta que queremos iniciar o Jupyter Notebook.

Se tudo ocorrer sem erro, o Emacs abrirar uma janela para a administrao dos notebooks.
